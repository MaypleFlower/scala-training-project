# Requirements
## Backend
* As a dev, I want to be able to model a Todo List, including the following properties:
  * Task
  * Description
  * Due date
  * Completion status
* As a dev, I want to be able to store Todo Lists in a database
## Frontend
* As a user, I want to be able to interface with an API to:
  * Create a Todo List
  * Retrieve a Todo List
  * Retrieve all Todo Lists
  * Edit a Todo List, including
    * Task
    * Description
    * Due Date
    * Completion status
  * Delete a Todo List
  * Delete all Todo Lists
* As a user, it would be nice to have a UI to navigate the whole process of available API endpoints more easily
